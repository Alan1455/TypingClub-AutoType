# TypingClub-AutoType

一個受 TypingClub.com 啟發的本地自動打字與練習腳本  
⚠️ 本程式僅供**學習、開發與本地測試使用**，請勿用於任何線上平台或作弊行為

## ✨ 功能特色
- 可輸入目標 WPM (預設 120，最高 150)
- 模擬自然的人類打字延遲 (隨機間隔)
- 可自動進行段落或練習測試 (選擇性)
- 適合用於開發打字練習工具、模擬人類鍵盤輸入或測試 key event 行為

## 🧠 運作原理
此腳本會:
1. 讀取頁面上的目標字元
2. 根據設定的 WPM 模擬輸入延遲
3. 在完成一段後自動下一關

## ⚙️ 使用方式
1. 將 JavaScript 程式碼貼到網站的Console
2. 執行後輸入目標 WPM
3. 程式會根據設定速度模擬輸入
